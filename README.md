# sh4wn Portfolio Website

This is a complete recreation of the portfolio website at https://willing-knowledge-231651.framer.app/about

## Project Structure

```
sh4wn-portfolio-website/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # All CSS styles
├── js/
│   └── script.js       # JavaScript for interactivity (FAQ)
├── assets/
│   └── images/         # All images from the website
└── README.md           # This file
```

## Features

- **Exact recreation** of the original Framer website
- All fonts, styles, sizes, and colors preserved
- All images downloaded and included locally
- Responsive design matching the original
- Interactive FAQ section
- Navigation with active state
- Testimonials carousel section

## Fonts Used

- **Instrument Sans** (from Google Fonts) - Used for navigation, testimonials, and body text
- **Inter Variable** (from Google Fonts) - Used for headings and larger text

## Colors

The site uses the following color scheme:
- White (#fff)
- Light gray (#f7f7f7)
- Gray border (#d1d3d6)
- Gray text (#6c7179)
- Dark gray (#1f1f1f)
- Black (#111)

## Sections

1. **Navigation** - Fixed top navigation with links
2. **Hero Section** - Main title and testimonials carousel
3. **Features Section** - Date and bio information with image
4. **About Section** - Personal statement and background image
5. **Major Works Section** - Grid of project cards
6. **FAQ Section** - Expandable FAQ items
7. **Footer** - Copyright and closing message

## Usage

Simply open `index.html` in a web browser. All assets are included locally, so no server is required for basic viewing.

## Notes

- The website uses Google Fonts which require an internet connection to load
- All images have been downloaded and stored locally
- The FAQ section includes interactive JavaScript for expanding/collapsing items
